ImportOptions Class Reference

#include <lomse_import_options.h>

Detailed Description

Class ImportOptions stores and manages the settings for dealing and fixing malformed imported files. This class is a singleton and the only instance is stored in the LibraryScope object. You can access this object and modify its settings either from LomseDoorway object or from the LibraryScope object:

LomseDoorway* pLomse = ...
ImportOptions* opt = pLomse->get_import_options();
LibraryScope& lomse = ...
ImportOptions* opt = lomse.get_import_options();

Example of use:

LomseDoorway* pLomse = ...
ImportOptions* opt = pLomse->get_import_options();
opt->fix_beams(true);
Presenter* pPresenter = pLomse->open_document(ViewFactory::k_view_vertical_book,
"my_score.xml");
...
opt->fix_beams(false);
Presenter* pPresenter = pLomse->open_document(ViewFactory::k_view_vertical_book,
"other_score.xml");

Settings description and default values for Lomse library are described in the following table:

Setting Default value Description
fix_beams false When true, if beam information is not congruent with note type, the importer will fix the beam.
use_default_clefs false When true, if an score part has pitched notes but the clef is missing, the importer will assume a G or an F4 clef, depending on notes pitch range.
