typedef void(* pt2NotifyFunction)(void *, SpEventInfo)

A callback function for receiving Lomse events . The callback funcion will receive two arguments:

  • The first one is the pointer your application passed to Lomse when setting the callback, normally a pointer to the instance of the object that will receive the notificatios or a NULL pointer for static methods or for C functions.
  • The second argument is the event to process.

See How Lomse callbacks work.

typedef void(* pt2RequestFunction)(void *, Request *)

A callback function for receiving Lomse requests. The callback funcion will receive two arguments:

  • The first one is the pointer your application passed to Lomse when setting the callback, normally a pointer to the instance of the object that will receive the notificatios or a NULL pointer for static methods or for C functions.
  • The second argument is the requests to process.

See How Lomse callbacks work.
